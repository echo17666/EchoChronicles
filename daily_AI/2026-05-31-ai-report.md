### OpenAI 推出 Rosalind Biodefense 生物防御计划
**发布时间：** 2026年5月29日
**发布公司：** OpenAI

### OpenAI 实时翻译模型亮相智能眼镜场景
**发布时间：** 2026年5月30日
**发布公司：** OpenAI
**详细内容：** OpenAI 联合创始人 Greg Brockman 在社交平台展示了 gpt-realtime-translate 模型在智能眼镜上的实际运行效果。该专用语音翻译模型支持 70 多种输入语言实时翻译为 13 种输出语言，强调大语言模型虽强，但特定用例仍需专用模型。OpenAI 于 5 月初已正式发布 GPT-Realtime-2、GPT-Realtime-Translate 和 GPT-Realtime-Whisper 三款实时语音 API 模型。
**来源：** Greg Brockman X账号 / AI HOT日报

### 阶跃星辰开源 Step 3.7 Flash 模型
**发布时间：** 2026年5月29日
**发布公司：** 阶跃星辰（StepFun）
**详细内容：** 阶跃星辰发布并开源面向生产级 Agent 场景优化的 Step 3.7 Flash 模型。该模型采用 198B 参数 MoE 混合专家架构，仅约 11B 动态激活参数，原生支持 256K 超长上下文窗口。在 ClawEval-1.1（67.1 分）和 SimpleVQA Search（79.2 分）评测中排名第一，τ²-bench 工具使用可靠性得分超 98%。模型兼容 Claude Code、MCP 协议等工具链，权重以 Apache 2.0 许可开源。
**来源：** IT之家 / 澎湃新闻 / 钛媒体
