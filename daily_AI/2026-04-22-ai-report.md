### ChatGPT Images 2.0

**发布时间：** 2026年4月21日
**发布公司：** OpenAI
**详细内容：**
OpenAI正式发布新一代图像生成模型 ChatGPT Images 2.0（亦称 GPT Image 2.0）。该模型被定位为图像生成领域的重大升级，核心亮点包括：

- **双模式运行**：提供 Instant（快速）和 Thinking（推理）两种模式。Instant 模式侧重速度；Thinking 模式在生成前进行推理，可保持角色一致性、生成连贯的多场景叙事，适用于漫画创作、故事板设计等。
- **交互式图像工作流**：用户可通过对话方式迭代修图，模型在编辑过程中保留上下文，支持放大、调整元素、改变构图等操作，无需重新生成。
- **文本渲染大幅提升**：显著改善非拉丁文字的渲染效果，包括中文、日文、韩文、印地语和孟加拉语等，解决了此前图像模型中的文字乱码问题。
- **高分辨率与灵活比例**：支持最高 2K 分辨率输出，单次可生成最多 8 张图像，支持 3:1 到 1:3 的灵活宽高比。
- **联网与知识更新**：可调用 ChatGPT 的推理能力搜索互联网获取最新信息，知识截止日更新至 2025 年 12 月。

该模型已向全球 ChatGPT 和 Codex 用户开放，付费订阅用户可使用更强版本，同时通过 fal 等平台提供企业级 API。

### Claude Opus 4.7

**发布时间：** 2026年4月21日
**发布公司：** Anthropic
**详细内容：**
Anthropic 宣布 Claude Opus 4.7 全面上市（generally available），这是 Opus 4.6 后的重要版本升级。主要改进包括：

- **软件工程能力显著增强**：在 SWE-bench Verified 上得分 87.6%，GPQA 得分 94.2%，在最难的编程任务上表现优于前代，用户反馈可将此前需密切监督的复杂编码工作放心交给该模型。
- **视觉能力升级**：支持 3.3 倍更高分辨率的图像识别，100万 Token 超长上下文窗口。
- **新增 xhigh  effort 级别**：为复杂任务提供更高的推理深度。
- **网络安全 safeguard**：作为上周公布的 Project Glasswing 计划的首个落地模型，Opus 4.7 内置自动检测和阻断高风险网络安全滥用的防护机制。合法安全研究用途可通过新的 Cyber Verification Program 申请。
- **全平台可用**：已上线 Claude 全系列产品、API、Amazon Bedrock、Google Cloud Vertex AI 及 Microsoft Foundry。API 定价维持不变：输入 $5/百万 Token，输出 $25/百万 Token。开发者可通过模型标识符 `claude-opus-4-7` 调用。

**来源：** Anthropic 官方发布说明（releasebot.io）、Claude API 文档、Claude Code GitHub Releases
