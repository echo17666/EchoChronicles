### Meta 发布 Muse Spark 全新大语言模型

**发布时间：** 2026年4月28日
**发布公司：** Meta
**详细内容：**
Meta宣布推出 Muse Spark，这是由 Meta Superintelligence Labs 构建的全新大语言模型系列的首款产品。Muse Spark 标志着 Meta AI 从独立聊天工具向平台原生智能的转变。

核心亮点：
- 支持复杂推理任务，同时保持轻量化和快速响应
- 引入 Instant（快速）和 Thinking（深度思考）两种模式，用户可按需选择
- 支持多模态输入，可解读图像、响应视觉提示
- 引入多智能体任务处理架构，可同时部署多个"子智能体"协作处理复杂查询
- 已集成至 Instagram、Facebook、WhatsApp、Messenger 及 Meta AI 眼镜

**来源：** Campaign Middle East

### OpenAI 与 Microsoft 结束独家协议，模型正式登陆 AWS

**发布时间：** 2026年4月28日
**发布公司：** OpenAI / Microsoft / Amazon
**详细内容：**
OpenAI 与 Microsoft 宣布修订合作协议，Microsoft 放弃对 OpenAI 模型的独家销售权。随后，OpenAI 模型迅速登陆 Amazon Web Services。

核心动态：
- OpenAI 前沿模型、Codex 编程助手及 Managed Agents 上架 Amazon Bedrock
- GPT-5.4 已在 AWS 上线，GPT-5.5 预计数周内推出
- Microsoft 仍保留"主要云服务商"地位，新产品优先在 Azure 上线
- Microsoft 将继续获得 OpenAI 收入分成直至2030年
- 此变更为 OpenAI 与 Amazon 此前 500 亿美元投资合作的落地举措

**来源：** 洛杉矶时报 / CNBC / TechCrunch

### 前 DeepMind 顶级研究员获11亿美元融资创办 Ineffable Intelligence

**发布时间：** 2026年4月27日-28日
**发布公司：** Ineffable Intelligence
**详细内容：**
由前 Google DeepMind 强化学习团队负责人 David Silver 创立的英国AI实验室 Ineffable Intelligence 宣布完成 11 亿美元种子轮融资，估值达 51 亿美元，创下欧洲最大种子轮融资纪录。

核心信息：
- 投资方包括红杉资本（Sequoia）、NVIDIA、Google 等
- 目标是构建不依赖人类数据的"超级学习者"（superlearner）
- 采用强化学习技术，让AI通过试错自主学习
- David Silver 曾主导开发 AlphaZero（围棋/国际象棋/将棋）
- 公司宣称若成功，这将是"堪比达尔文进化论"的科学突破

**来源：** TechCrunch / Bloomberg / Benzinga

### Google Gemini 替代 Google Assistant 登陆400万辆通用汽车
