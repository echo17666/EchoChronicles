### Grok Speech-to-Text API 与 Text-to-Speech API

**发布时间：** 2026年4月19日

**发布公司：** xAI (Elon Musk's AI company)

**详细内容：**
xAI正式发布了两款独立的语音API产品，标志着该公司进军企业级语音AI市场：

**1. Grok Speech-to-Text (STT) API：**
- 支持25种语言的实时和批量语音转录
- 具备高级功能：说话人分离、词级时间戳、逆文本规范化
- 支持12种音频格式
- 在电话实体识别基准测试中，错误率仅为5.0%，显著优于竞争对手ElevenLabs (12.0%)、Deepgram (13.5%)和AssemblyAI (21.3%)
- 在医疗、法律和金融等垂直领域表现尤为出色
- 价格：批量处理$0.10/小时，流式处理$0.20/小时

**2. Grok Text-to-Speech (TTS) API：**
- 提供5种表现性语音：Ara、Eve、Leo、Rex、Sal
- 支持20种语言
- 支持内联语音标签（如[laugh]、[sigh]、[emphasis]），开发者可精细控制语音表达
- 价格：$4.20/百万字符

**技术亮点：**
这两款API运行在与Grok移动应用、特斯拉车载体验和Starlink客户支持相同的基础架构上，已处理数百万次交互，具备高可扩展性和低延迟性能。

**行业影响：**
xAI此次进军语音API市场，将直接挑战ElevenLabs、OpenAI Whisper、Deepgram和AssemblyAI等现有厂商。对于已经使用xAI技术栈的开发者而言，这提供了一站式语音解决方案，可能降低集成复杂度和成本。
